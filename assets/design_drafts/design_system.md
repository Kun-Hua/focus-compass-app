# Focus Compass Design System

## 1. Visual Style
- **Keywords**: Cold, Focused, Minimalist (Cold Productivity Minimalist)
- **Philosophy**: Reduce cognitive load, emphasize content and data.

## 2. Color Palette
| Role | Color | Hex | Usage |
|:--- |:--- |:--- |:--- |
| **Primary** | Blue | `#3B82F6` | Main actions, active states, branding |
| **Success** | Green | `#10B981` | Achievement, completion, positive trends |
| **Warning** | Amber | `#F59E0B` | Alerts, near-deadline, caution |
| **Error** | Red | `#EF4444` | Failures, broken promises, critical errors |
| **Background** | Light Gray | `#F9FAFB` | App background |
| **Surface** | White | `#FFFFFF` | Cards, modals, panels |

## 3. Typography
**Font Family**: Inter

| Level | Size | Weight | Usage |
|:--- |:--- |:--- |:--- |
| **H1** | 32px | Bold | Page titles, major stats |
| **H2** | 24px | SemiBold | Section headers, card titles |
| **Body** | 16px | Regular | General content, list items |
| **Caption** | 14px | Regular | Secondary text, timestamps, labels |

## 4. Components

### Buttons
- **Radius**: 8px
- **Primary**: Blue background, White text.
- **Secondary**: White background, Gray border, Dark text.

### Cards
- **Radius**: 12px
- **Shadow**: Soft, diffuse shadow (e.g., `0 4px 6px -1px rgba(0, 0, 0, 0.1)`).
- **Background**: White (`#FFFFFF`).

### Inputs
- **Style**: Clean border, 8px radius.
- **Focus**: Blue border ring.

### Icons
- **Style**: Linear, simple, consistent stroke width.
